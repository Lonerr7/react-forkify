# React-Forkify

This repository is a complete version of React-Forkify. This project was built to train CRUD operations and http requests. It has also a local storage interaction.

---

## Used technologies:

- ReactJs
- Redux Toolkit
- SASS(SCSS)
- Redux-persist (localstorage)
